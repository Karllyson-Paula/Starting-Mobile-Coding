# MultiTool App - 2026 Edition 🚀

Este é um aplicativo utilitário versátil desenvolvido em React Native utilizando o ecossistema Expo. O objetivo é centralizar ferramentas produtivas e interativas em uma única interface fluida e intuitiva.
🌟 Funcionalidades Principais

    Hub de Produtividade: Relógio em tempo real, calendário mensal e calculadora funcional.

    Gestão de Notas: Sistema de anotações dinâmico com criação e exclusão.

    Contador Inteligente: Sistema de cliques com seleção de multiplicador de impacto.

    Easter Egg Interativo: Uma jornada lúdica entre um Donut e um Café através de navegação por cliques.

🛠️ Tecnologias Utilizadas

    React Native

    Expo

    React Navigation (Tab & Stack)

    Ionicons (Vetores de ícones)

🚀 Como Rodar

    https://snack.expo.dev/@bilolastico/multiutility-hub

    Aguarde o carregamento no simulador ou escaneie o QR Code com o app Expo Go no seu celular.
